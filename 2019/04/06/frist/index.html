首先，什么是机器学习呢？
机器学习说白了就是 Looking for a Function From Data 。举个例子：如果你想要识别所给图像是猫还是狗，就要通过机器找到一个Function，通过一开始所给定的数据，让他有判断没有见过的图像是猫还是狗的能力，如图。

Framework

这里我们来大致说一下如何找出解决问题的Function，主要分为三个步骤：A set of function , Goodness of function f , Pick The “Best” Function f*

举个例子现在要预测神奇宝贝进化之后的攻击力，我们要先设许多function: 如f(x) , g(x) , h(x)… x表示进化前的攻击力,函数的结果是进化后的攻击力 , 我们让

我们要根据许许多多神奇宝贝的进化数据，来判断每个function的好坏，最后用一个有效率的演算法，调出最好的function。

Learning Map

下面的图是一个大致的框架，

在图里面，蓝色的表示机器学习的种类。机器学习主要有监督学习、半监督学习、迁移学习、无监督学习、强化学习。在监督学习中，橙色表示我们可以用机器学习来做什么，比如回归，分类，结构化学习。绿色表示我们用什么方法去解决这些问题，比如用线性模型，深度学习，SVM…

监督学习：

  • 必须确定目标变量的值，以便机器学习算法可以发现特征和目标变量之间的关系。在监督学习中，给定一组数据，我们知道正确的输出结果应该是什么样子，并且知道在输入和输出之间有着一个特定的关系。
  • 训练样本 = 特征(feature) + 目标变量(label: 分类-离散值/回归-连续值)
    半监督学习：
  • 半监督学习处在监督学习和无监督学习之间，Semi-Supervised Learning中使用的数据，有一部分是标记过的，而大部分是没有标记的。因此和监督学习相比，半监督学习的成本较低，但是又能达到较高的准确度。

迁移学习：

  • 迁移学习能够将适用于大数据的模型迁移到小数据上，作为小数据模型的训练起点，节约训练神经网络需要的大量计算和时间资源。例如采用在计算机视觉挑战赛通过ImageNet数据（源数据）集训练出来的AlexNet 模型迁移应用到另一个新的数据集（目标数据集）重新进行训练（微调）。

无监督学习：

  • 无监督学习，即在未加标签的数据中，试图找到隐藏的结构。数据没有类别信息，也没有给定的目标值。
  • 无监督学习包括的类型：聚类：将数据集分成由类似的对象组成多个类。密度估计：通过样本分布的紧密程度，来估计与分组的相似性。
    此外，无监督学习还可以减少数据特征的维度，以便我们可以使用二维或三维图形更加直观地展示数据信息。

强化学习：

  • 所谓强化学习就是智能系统从环境到行为映射的学习，以使奖励信号（强化信号）函数值最大，由于外部给出的信息很少，强化学习系统必须依靠自身的经历进行自我学习。通过这种学习获取知识，改进行动方案以适应环境。
  • 状态学习最关键的三个因素：状态，行为，环境奖励。
